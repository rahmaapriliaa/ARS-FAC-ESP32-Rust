# ARS-FAC-ESP32-Rust
Adaptive Risk-Based Sampling and Fail-Safe Actuator Control (ARS-FAC) for ESP32-S3 using Rust. Simulates gas sensor input, classifies system risk levels, adjusts sampling intervals adaptively, and controls actuators (fan, LED, buzzer) in fail-safe mode. Includes data logging, visualization with GNUPlot, and WOKWI simulation examples.
